# Patitas Felices — Sistema para cuidar mejor

## Introducción
Patitas Felices crece todos los días, y con ese crecimiento aparecen pequeñas piedras en el camino: turnos que se pisan, historias clínicas que se guardan “por ahí”, agendas que no hablan entre sí. Este sistema nace para despejar el camino: una mesa ordenada donde conviven los datos de cada mascota, la agenda del equipo y la comunicación con las familias.

Más que “otro software”, es una forma de trabajo compartida. Una ventana única donde el cliente reserva turno, el recepcionista coordina, el veterinario registra la consulta y todos miran la misma foto actualizada.

## Propósito
Que la veterinaria respire mejor. Menos papeles sueltos, menos idas y vueltas, menos dudas. Una herramienta que:
- Da identidad a cada usuario y le muestra lo que necesita ver.
- Hace de la agenda un mapa claro: programar, mover o cancelar sin perder el hilo.
- Vincula turnos con mascotas y profesionales para que nada quede fuera.
- Convierte el historial clínico en una bitácora fácil de leer.
- Ayuda a asignar tareas según especialidad para que el equipo fluya.
- Ofrece al cliente un trato más claro y cercano.

## Qué hace
### Personas y roles
- Crea usuarios y define qué puede ver y hacer cada uno.
- Administra perfiles como Administrador, Veterinario, Cliente y roles operativos.

### Mascotas
- Registra datos clave (nombre, especie, raza, nacimiento, dueño).
- Las vincula directamente con su familia.

### Turnos
- Permite pedir, reprogramar y cancelar.
- Muestra disponibilidad y estado de cada turno.

### Profesionales
- Registra veterinarios y su especialidad.
- Asigna turnos según agenda y área.

### Historia clínica
- Guarda consultas, diagnósticos y tratamientos.
- Relaciona cada registro con su mascota y el profesional.
- Ordena por fecha para revisar rápido.

### Operación diaria
- Ofrece reportes básicos de turnos, pacientes y equipo.
- Conecta áreas para coordinar mejor.

## Por ahora, fuera de la cancha

## Cómo se vive
- Registro con confirmación por email y vencimiento del enlace.
- Acceso según rol y paneles pensados para cada tarea.
- Gestión de turnos desde un lugar central.
- Mascotas e historias clínicas siempre a mano.
- Secciones especiales para roles operativos (quirófano, laboratorio, farmacia, stock, etc.).
- Recuperación de contraseña con enlace temporal.
